# Даня - Data Collection & Generation

## Зона ответственности
- Сбор реальных исторических данных WWII
- Генерация синтетических изображений с помощью SDXL-Turbo
- Управление датасетами

## Файлы
- `data/download_real_wikimedia.py` - скрипт загрузки реальных фото
- `data/generate_synthetic_sdxl.py` - генерация синтетики
- `data/real/` - папка с реальными изображениями
- `data/synth/` - папка с синтетическими изображениями
