# Боря - Data Processing & Embeddings

## Зона ответственности
- Построение объединенного датасета
- Извлечение CLIP эмбеддингов
- Предобработка данных

## Файлы
- `src/build_manifest.py` - объединение манифестов и train/val/test split
- `src/extract_embeddings.py` - извлечение CLIP эмбеддингов
- `data/manifest.csv` - финальный манифест
- `data/embeddings.npz` - сохраненные эмбеддинги
